# 🛒 React E-Commerce

A modern and responsive E-Commerce web application built with React, featuring authentication, shopping cart, favorites, Stripe test payments, and a clean user experience.

## Live Demo

🔗 https://e-commerce-react-zeta-lac.vercel.app/

---

##  Features

- Authentication with Email & Password
-  Google Sign In
-  Browse Products
-  Product Search
-  Favorites
-  Shopping Cart
-  Dark / Light Mode
-  Fully Responsive Design
-  Contact Form
-  Fast and Smooth UI

---

##  Built With

### Frontend

- React
- React Router
- Redux Toolkit
- Tailwind CSS
- Shadcn UI
- React Icons

### Backend & Services

- Firebase Authentication
- Fake Store API

##  Responsive

The application is optimized for:

- Mobile
